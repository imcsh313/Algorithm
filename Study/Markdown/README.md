# Markdown 마크다운

- Markdown을 사용하면서 찾아본 문법을 기록합니다.
- 규칙은 없습니다.

1. 체크박스 check box
    ``` markdown
    - [ ] 으로 체크 박스를 만들 수 있습니다.
    - [x] x를 넣어서 확인 형태로 바꿀 수 있습니다.
    - [X] 대문자 X도 가능합니다.

    - [] 괄호 안에 여백이 없으면 안됩니다! 
    ```
